Welcome to FlashFinder, a powerful and intuitive file management and search utility designed by TerrinTech. This documentation aims to provide a comprehensive guide on how to effectively utilize the many features FlashFinder offers.

Introduction

FlashFinder is a modern, feature-rich file browser that combines quick file navigation with advanced search capabilities, making it easy to manage and locate your files.

Main Features

  • Quick Navigation: Navigate through your files seamlessly with the back, forward, and up navigation buttons.
  • Quick Select: Quickly select items by typing the first letter and cycle through files beginning with that letter.
  • Advanced Search: With an integrated search bar, FlashFinder offers rapid file searching across all drives or within specific directories.
  • Quick Access Sidebar: Easily access frequently used or pinned directories and files from the Quick Access Sidebar.
  • File Preview: Get a quick preview of selected files without having to open them.
  • Context Menus: Access various file operations with right-click context menus.
  • Undo/Redo: Made a mistake? Use the undo and redo shortcuts to revert actions.
  • Custom Themes: Enjoy a user interface that adjusts according to your system's theme - light or dark.
  • Status and Info Bar: Stay informed with real-time updates on file operations, indexing status, and more.
  • Tabbed Browsing: Browse multiple differnet paths in 1 FlashFinder window.

Navigation

At the core of FlashFinder is its navigation system. With the navigation toolbar, you can:

  • Go Back to previously viewed directories.
  • Move Forward if you've gone back.
  • Move Up one directory level.

The address bar also provides a breadcrumb-style navigation, allowing for quick jumps to any directory level.

The main window makes use of a tabbed browsing interface allowing for the viewing of multiple direcotries in 1 window.

  • Creating a new tab: Simply click the "+" button in the top right corner of the view.
  • Renaming a tab: To rename a tab, double click the tab title and then enter a new title.
  • Deleting tabs: Simply click the "X" button in the tab to close it.
  • Moving files between tabs: To move files between tabs, drag a file over the tab and it will open the tab after a short delay, then drop the file as necessary.

Search

FlashFinder's advanced search mechanism lets you:

  • Quickly search for files across all drives.
  • View search results in a dedicated dock.
  • Use regular expressions for complex search patterns.
  • Toggle between different search modes for more refined results.
  • Use commands for searching.

Refer to Advanced Searching for more information.

Quick Access & Pinned Items

The Quick Access sidebar provides rapid access to frequently used directories and files. You can also pin your favorite directories or files for even faster access.

The button to hide and show the quick access sidebar is on the left side of the bottom toolbar.

File Preview

Clicking on a file provides a quick preview in the dedicated Preview Dock. This is especially useful for getting a glimpse of file contents without the need for opening it in a separate application.

The button to open the preview is in the left side of the bottom toolbar.

  • Zoom: Ctrl + Scroll - Zooms to the cursor.
  • Horizontal scroll: Shift + Scroll - Scrolls the preview horizontally.

File Operations

FlashFinder offers a plethora of file operations right at your fingertips:

  • Copy, Cut, and Paste files or directories.
  • Quickly create new files or folders.
  • Delete files or move them to the trash.
  • Rename files with ease.
  • Move files to a new location.

Breadcrumbs Navigation

The Breadcrumbs Navigation system is a feature of FlashFinder that allows users to understand their current directory position and navigate swiftly through the filesystem hierarchy. It provides a visual representation of the directory path, where each directory is clickable, allowing users to jump directly to that directory.

Components

  1. Address Bar Widget: The primary widget that houses the breadcrumbs navigation system.
  2. Breadcrumb Button: Represents each directory segment in the breadcrumbs trail. Each button is clickable, and some can also accept drag-and-drop actions for files.

Functionalities

Switching Views

The BreadcrumbsBar widget offers two main views:

  • Breadcrumbs View: Represents the directory path as a series of clickable buttons.
  • Line Edit View: Represents the directory path as editable text. This view is useful for directly typing or pasting a path.

The user can switch between the Line Edit View and the Breadcrumbs View by clicking off a button or clicking the "..." button for long paths.

Navigation

Clicking on any segment (button) in the breadcrumbs trail will navigate the FlashFinder application directly to that directory.

Drag-and-Drop

FlashFinder supports drag-and-drop actions. When files are dragged onto a Breadcrumb Button, they can be moved to the represented directory, enhancing the user's ability to organize files effortlessly.

Optionally, FlashFinder aslo supports directly going to a path or launch a app/file in its defalt program when entered in the line edit mode.

Adjusting Breadcrumbs

As the application window is resized, or the directory depth increases, the breadcrumbs adjust dynamically. If the full path cannot be displayed due to space constraints, the Breadcrumbs Bar intelligently truncates the path and presents an ellipsis ("...") button. Clicking this button allows users to view and navigate to hidden parts of the path.

Status & Information Bar

The bottom toolbar of FlashFinder keeps you informed:

  • View the current status of file operations or searches.
  • Keep track of the number of indexed files for faster search results.
  • Get a quick count of files in the current directory.

Settings

The "General" settings tab provides a few options to customize FlashFinder.

  • Scroll To: sets whether the main view will scroll to the selected search result item automatically or not.
  • Replace Duplicates: sets whether you will be prompted to replace or rename the duplicate pasted file.
  • External Drop Mode: sets the way item drops from outside FlashFinder are handled. Options include pasting the file (Creating a new copy) or moveing the file (if possible, moves the file to the current path).

Permission Modes

  • Read Mode: In this mode, users have the ability to view files and directories but are restricted from making any modifications. This includes actions like editing, moving, renaming, or deleting files and directories.
  • Write Mode: When toggled to this mode, users gain the capability to make changes to the files and directories. This includes creating, editing, moving, renaming, and deleting operations.

Switching Permissions

The permission mode can be swiftly toggled using the "Access" button situated on the bottom toolbar of the FlashFinder application. This button displays the current mode - either "Read" or "Write". Simply clicking on this button will switch between the two modes, providing an intuitive mechanism for users to control their level of access.

Permission Warnings

If a user attempts an action that is not permitted in the current mode, FlashFinder will promptly display a warning message. For instance, if a user tries to edit a file while in "Read" mode, a notification will alert them about the lack of required permissions. This ensures that users are always aware of their current permissions and prevents unintentional modifications.

Keyboard Shortcuts

Efficiently navigate and manage your files with the following shortcuts:

  • Copy Item: Ctrl + C - Copies the selected item(s).
  • Copy Item Name: Shift + Alt + C - Copies the name of the selected item (including the file extension).
  • Copy Item Path: Ctrl + Shift + C - Copies the path of the selected item.
  • Copy Current Path: Ctrl + Alt + C - Copies the current directory path.
  • Paste Item: Ctrl + V - Pastes item(s) from the clipboard.
  • Move Item: Ctrl + M - Moves the selected item(s).
  • Rename Item: Ctrl + R - Renames the selected item.
  • Delete Item: Del - Moves the selected item(s) to trash.
  • New File: Ctrl + N - Creates a new file.
  • New Folder: Ctrl + Shift + N - Creates a new folder.
  • Change Access: Ctrl + Shift + A - Modifies access permissions.
  • Undo: Ctrl + Z - Undoes the last action.
  • Open Item: Return - Opens the selected item.
  • Search: Ctrl + F - Focuses on the search bar.
  • Go to Parent Directory: Backspace - Navigates to up a directory level.
  • Show Properties: Alt + Enter - Displays the properties of the selected item.
  • Add to Favorites: Ctrl + B - Pins the selected item to the sidebar.
  • Open Settings: Ctrl + I - Opens the settings pane.
  • Select All: Ctrl + A - Selects all items in the view.
  • Open a new instance Ctrl + Click on the title bar - Opens a new FlashFinder window.
  • New Tab: Ctrl + T - Create a new tab.
  • Quick Select: Any Character - Cycles thorugh items that start with the character.
