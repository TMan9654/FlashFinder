Getting Started with FlashFinder

Welcome to FlashFinder, a powerful file explorer replacement tailored for Windows. This guide will help you get started with the basic features and functionalities of the app.

Installation

To install FlashFinder:

  1. Navigate to the proper drive location "X:\2_Design Tools\8_Temp Terrin Part Time Work" and copy the FlashFinder desktop shortcut
  2. Paste the shortcut wherever is most convienent for your use.
  3. For convienence, you can right click the shortcut and click "Pin to Start" and this will add it to your start menu.
  4. Once the shortcut is pasted, launch FlashFinder from the either the desktop shortcut, start menu, or the taskbar (see how to pin to taskbar).

To pin FlashFinder to your taskbar:

  1. Navigate to the proper drive location "X:\2_Design Tools\8_Temp Terrin Part Time Work" and find the latest FlashFinder version folder.
  2. In the version folder, look for the executable "FlashFinder.exe" and copy this executable.
  3. Paste the executable on your desktop, this is just temporary.
  4. Right click the pasted executable and click "Create shortcut".
  5. At this time you can optionally rename the shortcut to the name you'd like to see displayed in the taskbar. Recommended name: "FlashFinder V#".
  6. Now right click the shortcut and click "Pin to taskbar".
  7. Now you can delete the "FlashFinder.exe" executable and shortcut from your desktop.
  8. Right click the taskbar shortcut then right click the name that's shown and select "Properties".
  9. At this time, navigate back to the original executable in steps 1 & 2, select the executable, then use the file explorer tool "Copy path" found in the "Home" tab to copy the path to the executable.
  10. Return back to the properties window opened in step 8 and paste the path in the "Target" entry box.
  11. In the "Start in" entry box, paste the same path but remove the executable name.
    12. Example:
    • "Target" = "X:\2_Design Tools\8_Temp Terrin Part Time Work\FlashFinder V7.2\FlashFinder.exe"
    • "Start in" = "X:\2_Design Tools\8_Temp Terrin Part Time Work\FlashFinder V7.2\"
  12. Now click "Apply" and then click "OK", the taskbar shortcut should now function properly.

Basic Navigation

The main interface is designed to be familiar to Windows users. Use the left pane to navigate between directories and the right main pane to view and manage files.

Use the top toolbar buttons to navigate through paths or use the address bar directly.

Right click items to access the context menu providing access to various commands.

Searching for Files

Use the search bar at the top to quickly find files or directories. To start a baisc search, simply enter the term you are trying to find and the search will return results containing your query.

FlashFinder's advanced search capabilities will be covered in the Advanced Searching guide.
