Power Rename Feature Guide

Back to Main Menu

The Power Rename feature provides an advanced mechanism for renaming files and directories, allowing users to efficiently perform batch renaming with a range of customization options.

Search and Replace

Search for: Enter the keyword or phrase you wish to search for within filenames.
Replace with: Enter the text with which you want to replace the searched keyword or phrase.

Options

  • Include All Items In Current Directory: When checked, Power Rename will apply the renaming operation to all items in the current directory.
  • Use Regular Expressions: Enables the use of regular expressions in the "Search for" field for advanced pattern matching.
  • Match All Occurrences: When checked, all occurrences of the search term within a filename will be replaced. Otherwise, only the first occurrence will be replaced.
  • Case Sensitive: Enables case-sensitive searching.

Apply To Options

Choose where you want the renaming operation to be applied:

  • Files: Only files will be renamed.
  • Folders: Only folders/directories will be renamed.
  • Both: Both files and folders will be renamed.

Additional Features

  • Include Subfolders: When activated, the renaming operation will also be applied to items within subfolders of the current directory.
  • Enumerate Selected Items: Allows for the sequential numbering of selected items. Useful for sorting purposes.

Text Formatting Options

Choose the desired text format for the renamed items:

  • All Lowercase: Convert all characters in the filename to lowercase.
  • All Uppercase: Convert all characters in the filename to uppercase.
  • Title Case: Capitalize the first letter of each word in the filename.
  • Capitalize Each Word: Similar to Title Case but ensures that every word is capitalized, even those like 'of', 'and', 'in', etc.

Preview Section

Before committing to the renaming operation, users can view a preview of the changes. This section displays the original filenames and their proposed new names, allowing users to verify the renaming actions before applying them.

Finishing Up

Once you have made your selections and configurations:

  • Click the OK button to apply the renaming operation.
  • Click the Cancel button to exit the Power Rename feature without making changes.
